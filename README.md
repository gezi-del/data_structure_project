# 代码解析
- # 1.Graph.py：地图生成文件
- ### Random Connected Weighted Graph Generator (RCWG)

  这是一个基于 Python 实现的**随机全连通带权无向图生成器**。该工具旨在为路径规划算法（如 A*、Dijkstra）、物流网络仿真或社交网络拓扑研究提供标准化的测试环境。

---

### 1. 核心特性

* **全连通保证**：利用贪心策略（类似 Prim 算法）确保生成的图中任意两点均可达。
* **地理空间模拟**：节点分布在二维欧几里得空间，边权自动根据物理距离计算。
* **拓扑可调性**：支持通过 `extra_k` 参数调节网络的稠密程度，模拟从“稀疏树状”到“复杂网状”的转变。
* **可视化集成**：内置基于 `Matplotlib` 的绘图引擎，支持不同类型节点（仓库、充电站、任务点）的颜色区分。
* **高效存储**：采用邻接表（Nested Dictionary）结构，在保证 $O(1)$ 查边效率的同时兼顾空间复杂度。

---

### 2. 算法实现原理

本工具生成图的过程分为三个阶段：

#### 阶段 I：节点散布 (Node Scattering)
在指定的 $[0, Width] \times [0, Height]$ 区域内随机均匀分布节点，并赋予唯一的 ID 和类型标签。

#### 阶段 II：强制连通 (Connectivity Guarantee)
采用类似 **最小生成树 (MST)** 的构建逻辑：
1.  初始化：随机选定一个起点加入“已连通集合”。
2.  迭代：寻找连接“已连通集合”与“未连通集合”之间欧氏距离最短的边。
3.  闭合：重复此过程直到所有点均并入同一连通分量。
> 这在数学上保证了图的连通性 $C = 1$。

#### 阶段 III：拓扑增强 (Extra Neighbors)
为了使图更接近现实中的道路网（具有冗余路径），对每个节点额外连接其最近的 $k$ 个邻居。这一步引入了环路，增强了网络的鲁棒性。
